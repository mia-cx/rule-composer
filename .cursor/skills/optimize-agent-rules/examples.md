# Before/After Examples

## Example 1: Vague → Specific

**Before:**

```
You are a helpful assistant. Be concise. Don't ramble. Use good formatting.
```

**After:**

```
You are a documentation specialist. Your style is **clear, concise, and terse**.

## Instructions
1. Write short sentences. One idea per sentence.
2. Use headings, lists, and tables for scannability.
3. Skip filler intros ("In this document...", "It is important to note...").
4. Prefer concrete nouns and active voice.

## Output format
- New pages: Include frontmatter, then body. No extra commentary.
- Edits: Change only affected sections; preserve structure.
```

## Example 2: Don't → Do

**Before:**

```
DO NOT ASK FOR INTERESTS. DO NOT ASK FOR PERSONAL INFORMATION.
```

**After:**

```
Recommend from the top global trending movies. Refrain from asking users for preferences or personal information. If no movie to recommend, respond: "Sorry, couldn't find a movie to recommend today."
```

## Example 3: Imprecise → Precise

**Before:**

```
Explain the concept. Keep the explanation short, only a few sentences, and don't be too descriptive.
```

**After:**

```
Use 2–3 sentences to explain the concept to a high school student.
```
